How to start Programming ?

First of all, you need to know that there is a difference between programming and programming languages. Programming is an art. Start your first language from a standard book and learn another language as required. The book helps you to learn deeply and clear your concepts. Learn one Programming language and become master in it. Choose any of the available programming languages which you will be learning. Choice of programming language may differ from person to person. I suggest you to start with a hard language so you can learn other languages easily. You can start with the CS50 course on edx. CS50 is an online introductory course on Computer Science taught at Harvard University. CS50
Learning by doing is a fun way of learning. By making projects you never get bored. Ask your doubts on Stack Overflow or Quora.
